# Livresgourmands
1.Description du projet:
LivresGourmands.net est une plateforme de commerce électronique spécialisée dans la vente de livres de cuisine adaptés à tous les niveaux (débutants, amateurs, chefs).
Objectifs du projet:
-Modélisation des besoins (Cas d'utilisation).
- Conception de l'architecture de données (Diagramme de classes).
- Mise en place de l'environnement de collaboration Git.
Technologies du projet :
Frontend : HTML, CSS, JavaScript, React
Backend : Node.js avec Express
Base de données : MySQL
